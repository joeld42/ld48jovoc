Strange Loop


Strange Loop is a puzzle game. The goal of the game is to create one continous loop that wraps around the whole shape. The touches every cell, but might not touch every corner dot. There are hints marked on the board, using those hints and some logical deduction, you can solve the whole loop!

This is inspired by games like Puzzle Loop aka Slitherlink or Loopy. Check out these games for more hints and strategies!

How To Play

When you start the game, some cells are marked with symbols.

  • A blank cell is unlabeled, and gives you no clues for the puzzle.
  • A cell with a CIRCLED X means none of the edges are part of the loop (but the corners may be)
  • A cell with one square has exactly one edge that is part of the loop
  • A cell with two square has exactly two edges that is part of the loop
  • A cell with three square has exactly three edges that is part of the loop
You may mark some edges with a red X, this is not required to solve it but it can help you keep track of which edges you know are empty. Some edges start out marked as part of the loop or as X'd out, these marking can't be modifed.

Controls

  • Drag the puzzle to rotate it
  • Click on an edge to cycle it between BLANK, MARKED, and X'd OUT
  • Shift-Click on a marked edge to make it blank
  • Shift-Click on a blank edge to make it X'd out
  • Press the "Check" button when you think you've solved the puzzle.

Cheat Codes

  • Press 0 to enable cheat codes.
  • Press 'D' to toggle overlay display of the solution
  • Press 'B' to fill in the edges with the correct solution

Strategy

It can be hard to figure out where to put the line. Here's an example:



Here, we see two cells with three marks next to each other. Since these can't form a closed loop (since the whole solution is a single loop), we know the open edge can't be between them, so they must make an S or a backwards S shape. Since we can see one marked edge on the top of the S, we can tell it's not a backwards S and can fill in the shape.

Next, since we've marked the top of the S and there is cell with one mark there, we can mark the other three edges on the one cell as open. That only leaves one direction for the loop to continue from the top of the S, so we extend it.

Now just keep doing that until you've solved the whole puzzle!
